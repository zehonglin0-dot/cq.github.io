第二阶段：店铺授权绑定

2.1 操作目的

将您的电商平台店铺授权给ERP系统，实现订单自动同步。

2.2 操作路径

设置 → 店铺管理 → 添加平台店铺
  • 选择平台
  • 输入店铺名称
  • 选择全球站点（非US）
  • 点击“前往授权”完成平台授权

2.3 状态确认

  • 启停状态：启用中
  • 授权状态：店铺已授权
⚠ 若显示未授权，订单将无法同步。

第三阶段：建立系统商品（SKU创建）

⚠ 所有发往海外仓的商品，必须提前在ERP建立SKU。

3.1 添加单品

商品 → 系统商品 → SKU → 添加SKU → 添加单品
填写信息 说明
SKU编码（必须唯一）建议与平台/仓库统一编码策略
商品名称清晰可识别（建议含系列/材质/颜色等）
商品条码（与SKU编码一致）用于扫描识别，建议直接=SKU
单品毛重（kg）必须真实
单品净重（kg）必须真实
单品尺寸（长×宽×高 cm）必须真实
重量与尺寸必须真实填写；该数据将影响运费计算与仓储计费。
  • 创建仓库商品 → 选择仓库 → 点击确定
  • 生成成功 → 下发仓库
必须截图：创建仓库商品 → 选择仓库 → 点击确定；生成成功 → 下发仓库 必填

第五阶段：商品入库流程

5.1 创建入库单

库存 → 入库订单 → 新增
  • 收货仓：选择对应账号名称
  • 添加商品 → 选择SKU → 填写数量

5.2 填写入库信息

  • 采购费用
  • 入库数量
  • 包装类型
原厂整箱包装：填写每箱数量、箱数、包装规格
混装说明：点击“混装”按钮；同一箱最多2个SKU；不同SKU必须有清晰物理隔离。
⚠ 不符合规范仓库有权拒收。

5.3 提交入库单

  • 点击保存
  • 点击提交
  • 状态显示“已下发”代表入库单成功提交
  • 复制入库单号并发送至业务对接群，以便仓库提前排仓

常见错误提示

问题 原因
订单未同步店铺未授权
订单无法出库未配对商品
运费异常SKU尺寸重量错误
仓库拒收未提交入库单或混装不规范
排查顺序建议：店铺授权状态 → 线上商品是否配对 → SKU尺寸重量 → 入库单是否已下发

账号创建 ↓ 店铺授权 ↓ 建立SKU ↓ 商品配对 ↓ 创建入库单 ↓ 平台开启海外仓 ↓ 订单自动审核
